# Paradox Team - موقع رصد الكويكبات

موقع متخصص في رصد الكويكبات والأجرام القريبة من الأرض، يقدم أدوات للبحث والتحليل والمشاركة المجتمعية.

## الملفات الرئيسية

### الصفحات الأساسية
- `index.html` - الصفحة الرئيسية
- `asteroids.html` - قائمة الكويكبات
- `asteroid-details.html` - تفاصيل كويكب معين
- `search-asteroids.html` - بحث متقدم عن الكويكبات
- `research.html` - الأبحاث العلمية
- `projects.html` - المشاريع الجارية
- `reports.html` - التقارير المنشورة
- `upload-report.html` - رفع تقارير جديدة
- `dashboard.html` - لوحة التحكم
- `profile.html` - الملف الشخصي
- `login.html` - تسجيل الدخول
- `join.html` - انضم للفريق
- `about-us.html` - من نحن
- `contact-us.html` - تواصل معنا

### الملفات المساعدة
- `styles.css` - التصميم الرئيسي
- `app.js` - البيانات والدوال المشتركة
- `css/dashboard.css` - تحسينات للوحة التحكم
- `js/profile.js` - وظائف الملف الشخصي

## كيفية التشغيل

1. قم بتنزيل جميع الملفات في مجلد واحد
2. افتح `index.html` في متصفح الويب
3. للتجربة، استخدم بيانات الدخول التجريبية:
   - البريد: `admin@paradox.local`
   - كلمة المرور: `secret`

## المميزات

- تصميم عصري مع خلفية فضائية
- دعم اللغة العربية (اتجاه من اليمين لليسار)
- نظام رصد وتتبع الكويكبات
- رفع وإدارة التقارير العلمية
- ملف شخصي للمستخدمين مع إحصائيات
- لوحة تحكم إدارية
- تصميم متجاوب مع جميع الأجهزة

## التقنيات المستخدمة

- HTML5
- CSS3 (متغيرات CSS، Grid، Flexbox)
- JavaScript (Vanilla)
- تصميم Material Glassmorphism

## التطوير المستقبلي

1. إضافة قاعدة بيانات حقيقية
2. نظام مستخدمين كامل
3. API للاتصال بقواعد بيانات الكويكبات العالمية
4. نظام محاكاة تفاعلي
5. تطبيق موبايل
